Git

Git is responsible for whatever happens in the github. We can say its the heart of githib To use git on the command line you need to download, install and configure it. You can also install Github CLI to use github from the command line.

About Remote repositories

A remote URL is Git's fancy of saying the place where your repositories are stored. You can only push two types of URL addresses which are an HTTPS or SSH URL's. You can use git remote set-url to change the remote URL
